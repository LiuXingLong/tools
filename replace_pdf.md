# replace_pdf.py — PDF 文字替换工具

使用 PyMuPDF (fitz) 在 PDF 中搜索并替换指定文字。

## 依赖

```bash
pip install pymupdf fonttools
```

## 功能

在 PDF 中查找指定文字，用白色矩形覆盖原文字后，在相同位置写入新文字。

## 使用方法

```bash
# 基本用法：通过 -r 参数指定替换对
python3 replace_pdf.py 单车业务报批_20260507154704.pdf 单车业务报批_20260507154704_output.pdf \
  -r 湘DD09478 湘DF09156 \
  -r 常宁龙泉寺 陈邢西 \
  -r 18073118839 18666361806 \
  -r 非营业企业 家庭自用 \
  -r 183800.00 127800.00 \
  -r 2021-06-11 2021-06-21 \
  -r 2026-06-06 2026-06-02 \
  -r 2027-06-05 2027-06-01


# 省略输出路径时，默认生成 输入文件名_output.pdf
python3 replace_pdf.py 单车业务报批_20260507154704.pdf \
  -r 湘DD09478 湘DF09156 \
  -r 常宁龙泉寺 陈邢西 \
  -r 18073118839 18666361806 \
  -r 非营业企业 家庭自用 \
  -r 183800.00 127800.00 \
  -r 2021-06-11 2021-06-21 \
  -r 2026-06-06 2026-06-02 \
  -r 2027-06-05 2027-06-01

# 从 JSON 文件读取替换映射
python3 replace_pdf.py 输入.pdf 输出.pdf -j replacements.json

# 指定页码和字号
python3 replace_pdf.py 输入.pdf 输出.pdf -p 1 -f 11 -r 旧文本 新文本
```

### 参数说明

| 参数 | 说明 |
|------|------|
| `input_pdf` | 输入 PDF 文件路径（必填） |
| `output_pdf` | 输出 PDF 文件路径（可选，默认: `输入文件名_output.pdf`） |
| `-r` / `--replacements` | 替换对，格式: `旧文本 新文本`，可重复使用 |
| `-j` / `--json` | 从 JSON 文件读取替换映射（格式: `{"旧": "新"}`） |
| `-p` / `--page` | 目标页码，从 0 开始（默认 0） |
| `-f` / `--fontsize` | 替换文本字号（默认 9） |

### JSON 文件格式示例

```json
{
  "湘DD09478": "湘DF09156",
  "常宁龙泉寺": "陈邢西",
  "18073118839": "18666361806",
  "非营业企业": "家庭自用",
  "183800.00": "127800.00",
  "2021-06-11": "2021-06-21",
  "2026-06-06": "2026-06-02",
  "2027-06-05": "2027-06-01"
}
```

## 工作原理

1. 使用 `page.search_for()` 搜索所有目标文字的位置矩形
2. 绘制白色矩形覆盖原文字
3. 在相同位置用宋体（SongtiSC，从 macOS Songti.ttc 提取）插入新文字
4. 字体大小和页码可通过参数自定义

## 注意事项

- 仅支持简单替换（覆盖 + 重写），无法保留原文字的字体、大小、样式
- 替换文字与原文字长度不同时可能出现对齐问题
- PDF 中 CID 编码的文字无法直接搜索（如某些中文字体），此脚本仅适用于可搜索文字
